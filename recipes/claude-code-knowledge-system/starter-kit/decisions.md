# decisions — 가벼운 결정 로그

> 한 화면 / 한 기능 범위의 가벼운 결정을 한 줄씩 기록한다. "왜 이 동작을 택했나"가 핵심.
> 되돌리기 쉬운 것만 여기. 모듈 구조·아키텍처처럼 되돌리기 힘든 결정은 `docs/adr/`로.
> 복사 위치: `docs/wiki/decisions.md` · 로딩: 온디맨드

## 적는 기준

- ✅ 적는다: 다른 합리적인 사람이라면 다르게 정할 수도 있던 **비자명한** 선택.
- ❌ 안 적는다: 당연한 것, 1회성 작업 내용.
