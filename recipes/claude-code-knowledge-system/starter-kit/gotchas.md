# gotchas — 반복 실수 패턴과 방지 규칙

> 작업 중 또 밟은 함정을 `틀림 → 맞음`으로 1줄씩 쌓는다. 단순 기록이 아니라 **다음 작업의 품질을 높이는** 참고 자료다.
> 복사 위치: `docs/wiki/gotchas.md` · 로딩: 온디맨드(작업 전 검색될 때만 읽힘)

## 적는 기준 (Merit Gate)

- ✅ 적는다: 6개월 뒤 *다른* 일을 할 때도 또 만날 함정. 비자명한 실수.
- ❌ 안 적는다: 이번 작업에만 해당하는 1회성 사항, 단순 작업로그.
- 새 항목 추가 전 유사 항목 확인 → 있으면 추가가 아닌 **업데이트**.
