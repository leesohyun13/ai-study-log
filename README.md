# ai-study-log

> 재밌게 떠들고 끝이 아니라, **레시피가 쌓여서 복리로 굴러가게.**
> 결과 자랑이 아니라 과정(레시피)을 통째로 공유하고, 못한 것도 솔직히 까본다.

개발자 · 마케터 · 기획자 4인의 AI 스터디 공동 지식 베이스.
2주에 1회 모여 서로의 AI 활용법을 공유하고 기록한다.

## 목표

- AI 업무 자동화 능력 ↑
- AI 활용 능력 ↑
- 미래 위험 대비

## 폴더 구조

```
recipes/    레시피 카드 (각자의 자동화·프롬프트 노하우)
sessions/   회차별 기록
trends/     동향 + 위험 점검 로그
templates/  레시피 카드 등 템플릿
```

## 규칙

1. 업무에서 AI로 뭔가 했으면 → `recipes/`에 카드 1장 남긴다.
2. **실패도 반드시 기록한다.** 거기서 제일 많이 배운다.
3. 결과만 적지 말고 **레시피 전부** — 남이 그대로 따라 할 수 있게.

## 새 레시피 작성하기

`templates/recipe.md`를 복사해 `recipes/`에 넣고 채운다.
